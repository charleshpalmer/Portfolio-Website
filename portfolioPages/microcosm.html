Microcosm VR - Game

An Puzzle/Parkour VR Game developed in Unity

About the Application

Microcosm is a VR Puzzle/Parkour game, that allows the player to explore a vibrant world created with lowpoly assets, and have remarkable freedom of movement, and idea driven gameplay.

Microcosm VR was created in Unity with C# being used to code the scripts that enabled the functionality during gameplay.

This application was my first experience in creating a video game on Unity, and using the Occulus XR SDK allowed me to experiment with the mathmatics behind physics and player movement.

Technologies Used

Unity, Occulus XR SDK, Office 365 Planner, GIT, C#.
