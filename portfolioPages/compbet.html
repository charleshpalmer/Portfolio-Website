COMPBET - Website

An FAQ, Account Management & Livechat Application

About the Application

The application provides a platform to discuss gambling, sports betting and matched betting in a safe and protected enviroment.

The website has FAQ's on the most popular games, these being; Blackjack, Roulette and Texas Hold'em, and offers users the oppertunity to create an account and use the livechat service to discuss their experiences and thoughts.

The application was created as a project to improve my knowledge of Back-end Technologies such as Node.js, MongoDB and Socket.io. It also provided me with the opertunity to practice Front-end design and implement functionality via Bootstrap libraries and styling with CSS.

Technologies Used

HTML, CSS, Bootstrap, NPM, GIT, Node.js, Express.js (w/ Flash), Passport.js and Socket.io
